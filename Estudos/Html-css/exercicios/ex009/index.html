Trabalhando com Listas

Listas ordenadas

Podemos criar listas ordenadas usando a tag <ol>(ordered list) e dentro dela criar nossa lista com a tag <li>(list item), também podemos difinir o tipo dela com a classe "type" onde podemos inserir a ordem em números romanos passando o parametro "i", ordem alfabética passando o parametro "A" e etc.... Podemos definir o começo da lista passando o parametro de início na classe "start"

  1. Acordar
  2. Tomar café
  3. Escovar os dentes
  4. Estudar
  5. Almoçar
  6. Treinar
  7. Estudar mais

Listas não ordenadas

Para fazer listas não ordenadas podemos usar a tag <ul>(unordered list) e dentro dela criar nossa lista com a tag <li>(list item) também podemos difinir o tipo dela com a classe "type" onde podemos passar o parametro "disc","square","circle"

  • Acordar
  • Tomar café
  • Escovar os dentes
  • Estudar
  • Almoçar
  • Treinar
  • Estudar mais
